Всем привет! Это мой проект, посвященный A/B тестированию.

Данный проект является финальной работой по окончанию курса "Аналитик данных -  karpov courses".

Мой проект состоит из трех частей:

В первой части был проведен AB-тест

Во второй части мне предстояло выполнить два SQL-запроса к БД:

-первый запрос вывел необходимый по заданию KPI, отображающий количество активных студентов, обучащихся на различных курсах платформы
-второй запрос вывел ключевые метрики для плаформы, а далее по полученным показателям (CR, ARPU, ARPAU, CR_active_users, CR_math_active_users) были построены графики, по которым видно, что значение метрик у тестовой группы везде выше, чем у контрольной

В третьей части проекта представлены две функции:

-первая функция считывает новые данные с яндекс-диска по API, присоединяет к имеющимся и пересчитывает метрики заново
-вторая функция строит три графика по различным метрикам (CR, ARPU, ARPPU)
